Pet Wars is a competency test to show understanding of basic object orientated principles. This is my response to test. 

## Install

Via Composer (if it was on packagist)

``` bash
$ composer require divarsoy/petwars
```

## Usage

``` php
$cat = new petwars\Cat();
$dog = new petwars\dog();
$cat->attack(dog);
```

## Testing

``` bash
$ phpunit
```
